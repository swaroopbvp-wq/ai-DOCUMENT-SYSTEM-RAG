# AI Document Intelligence System

A document question-answering system built using FastAPI, FAISS, and Sentence Transformers.

Users can upload documents and ask questions about their content.

## Features
- Upload TXT, PDF, DOCX documents
- Semantic search using vector embeddings
- FAISS vector database
- Ask questions about uploaded documents
- Simple web UI

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- FastAPI

AI / Retrieval:
- Sentence Transformers
- FAISS

Document Processing:
- pdfplumber
- python-docx
